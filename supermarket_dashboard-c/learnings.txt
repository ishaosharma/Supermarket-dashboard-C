MVT stands for:
Model: Handles data (loading CSV, cleaning, calculations)
View: Business logic (what to show, how to process)
Template: UI (Streamlit interface that users see)

Think of it like a restaurant:
Model = Kitchen (prepares food/data)
View = Waiter (decides what to serve)
Template = Dining area (where customers eat/see results)


What each library does:

numpy: Fast math operations on arrays
matplotlib: Creating charts (bar, line, pie)
streamlit: Building web interface
